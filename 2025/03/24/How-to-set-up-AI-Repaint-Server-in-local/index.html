How to Set Up AI Repaint Server Locally

This article will teach you how to set up an AI repaint server, so you can use your own computer as an AI server to perform AI repainting.

Mac currently does not support “Realistic” style transfer

Step 1: Get Civitai API Key

Some models need to be downloaded from Civitai and require an authorization token. You need to get one from https://civitai.com/user/account.

Just scroll down to the “API Keys” section and click “Add API key” to generate one. Save it for the next step.

Step 2: Install Python Environment

I have coded a Python script to run the installation automatically, so you need to make sure your computer can run Python.

Just verify that after running the command python --version in your computer’s terminal, you can get the result.

Step 3: Set Up & Run the Python Code

  • Copy the Python script initial_AI_repaint_server.py

  • Fill in your API key in the line civitai_api_key = ""

  • Move the script to the location where you want to build the server (like “Downloads”). It will take about 20GB of space.

  • Run the script python initial_AI_repaint_server.py in the terminal.

When the output is finished, in the results, you can find a log that looks like this:

That is your server address! Any time you want to use AI-Repainting, just run python initial_AI_repaint_server.py and keep the terminal open. Your computer will repaint the images for you.

PS: First time need to download some extra models, so may take a while.
