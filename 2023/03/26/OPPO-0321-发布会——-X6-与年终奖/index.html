OPPO 0321 发布会—— X6 与年终奖

讲讲发布会和年终奖

    很久没好好的写一些东西了。其实往前推一段时间，应该是从 OPPO 离职之后就很少有写长文的一些动力。去年离职后这小半年，发生了很多很多很很多的事，现不在这里引入了，等事情都结束了再说吧。

    回到主题，事情的起因是我在前一段时间收到了 OPPO 官方发过来的发布会邀请。当然，是把我当做了消费者的身份，也就是 O 粉进行邀请的。当我刚毕业进入 OPPO 的时候，因为很多需求都通不过，就和同事半开玩笑的注册了一个小号，打算在社区养大了假装马甲反馈来倒推需求通过。谁曾想到竟然因为这个原因，我的小号就被标记为了核心粉丝。

    话说回来，也是因为这个原因，我能有幸以一个前员工+O粉的身份参加 OPPO 的发布会，从台下见到张老板——这个之前需求展示碰到过好几次的男人。

写写手机

凭良心讲，X6 很不错。

大量的篇幅，都在讲手机的影像系统，讲暗光长焦的部分。OPPO 这次的确很舍得下血本，三颗摄像头都使用了一样的 CMOS ，……

印象很深的一页，你以为三张图片都是精心拍摄的，画面一转发现是一张图上扣下来的，配上顶部的“6倍长焦”，这种震慑力不言而喻。

以及这个精心设计的水印，我非常喜欢，哈苏的黑白 Logo 与标志性的橙色没有喧宾夺主，但是在低调中能看出一丝奢华感。

对了，不得不说这次发布会的设计，顶部的灯光应该是专门设计过的，也用了这个很经典的橙，并加上了相机圈作为设计元素。

这个从宣发到 ID 到软件的一脉相承的设计，的确很让人舒服，能感受到 OPPO 这个企业由内而外散发的一种贵族气质。

然后是除了相机的部分，我没有详细统计，不过根据主观感受来看，手机介绍部分应该有3/4 的篇幅在讲相机，剩下就只讲了互联、车机相关的功能。

/

说来也巧，这些都是之前待过的模块，也是很高兴之前提的一些需求终于能被实现了。

总之，Find X6 应该是一款很让人惊喜的产品，不仅补齐了 OPPO 之前“相对孱弱”的拍摄模块（好想吐槽马里亚纳但是涉及到保密协议还是算了），还往前领先了一大截。

我甚至有种感受——当然欢迎打脸——X6 之后手机影像很难再有大的突破，手机的尺寸就摆在那里，若要保证长焦、主摄、广角都使用大底的 CMOS ，X6 应该就是极限了。

均衡的水桶机再加上相机、互联方面的亮点，以及在安卓第一梯队的动效设计，我的建议是，有换机需求的完全可以无脑入。

写写年终奖

不过，与发布会的盛况之相对应的是，前同事群里的一些哀嚎。

今天刚好是大部分部分“开奖”的日子，也就是会告知今年年终奖的具体数额。根据群里已经开了的人来说，基本在3折左右。而且是在去年基础上的3折。

这里先告知一个前提，OPPO 是低 Base 浮动年终的公司。月薪如果如果是在深圳，是勉强够生活的水平，但与之相对应的，年薪则是可以达到十几甚至几十个月。

不过这也并非强制，因为之前员工可以选择固定新增，也就是高 Base ，固定月份年薪，也就是和正常公司一样。

但是在去年，公司整体薪酬改革，固定制也强制转为了“新浮动制”，明升、暗降。这个动荡集中体现在了今年年终奖上，固定制的也会被打折，而固定年终本来就很少，再次打折，夸张的甚至仅有四位数。

所以今天发布会到后期我其实很分裂🤯，一方面周围的环境的确让人情绪很高涨，每次 Pete 出一个样片观众就欢呼一次。另一方面随着群里不断有人开奖，哀嚎声也是此起彼伏。

（甚至开玩笑说要不要拿瓶水上去复刻一下宏颜获水名场景）

其他

Anyway ，随便写写，今天也是过去了，Find X6 作为我在 OPPO 经历过的最后一个项目，也算见证了最后的样子，希望大卖吧
